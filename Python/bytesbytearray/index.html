    bytes¶

    不可变序列

    bytearray¶

    字节数组,可变

    编码和解码¶

    字符串按照不同的字符集编码encode返回字节序列bytes

    encoding(encoding='utf-8',error= 'strict'') ----> bytes

    字节序列按照不同的字符集解码decode返回字符串

    bytes.decode(encoding='utf-8',error='strict') ----> str

    bytearray.decode(encoding='utf-8',error='strict') ----> str
