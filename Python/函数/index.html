    函数¶

    函数是一个具体功能的实现

    分类¶

    • 自定义函数
    • 内建函数

    定义¶

    def 函数名(参数列表):
    函数体(代码块)
    [return 返回值}

    使用¶

    使用函数名() 来调用一个函数.

    参数¶

    函数在调用的时候,会从外部传入一些变量,称之为参数.

    函数可以作为函数的形参以及实参,以此为基础出现了函数嵌套和装饰器.

    形参¶

    定义函数时,函数名后面的括号中的参数,称为形式参数,简称形参.

    默认值¶

    定义函数的时候给形参一个默认值,当传参时没有给定形参具体实参,则会自动使用默认值,可以防止函数报错.

    形参的分类¶
    位置参数¶

    通过位置来传递的参数,按照实参的传入顺序来得到具体的值.

    关键字参数¶

    通过关键字来传递,可以不按顺序传入

    keyword-only参数(Python3引入)¶

    当一个关键字参数后又出现一个位置参数,这个位置参数会变成keyword-only参数,

    当keyword-only参数出现时,传参需严格按照位置及关键字传参

    如果定义了keyword-only参数,则必须传参,否则报错

    不建议使用

    可变参数¶
    • 可变位置参数

    通过*args来接收,可以匹配任意多个位置参数

    • 可变关键字参数

    通过**kwargs来接收,可以匹配任意多个关键字参数

    注意¶

    定义函数时,为了后续使用,除非极其明确作用,否则建议使用默认值

    形参使用宗旨,简单的往前方,至于什么是简单的,看着简单的就是简单的.

    实参¶

    在函数调用阶段实际传入的参数称之为实参.

    函数的返回值¶

    • 所有函数都有返回值,没写隐式调用return None
    • 一个函数可以存在多个return语句,但是只执行第一个执行的return.
    • 如果函数执行了return语句,函数就会返回,当前被执行的return后面的语句就不会被执行
    • 作用:结束函数调用,返回值
    • return[1,2,3] 指明返回一个列表

    • return 1,2,3 看似返回多个值,实则被封装成了一个元组

    • fn._default_ 查看缺省值

    • fn._doc_ 返回函数的注释说明

    函数的嵌套¶

    首先调用内部局部作用域的值,没有再向外层寻找变量名

    外层作用域再内层作用域可见

    内层作用域inner中,如果定义了o = 97 相当于当前作用域中重新定义了一个新的变量o,但是这个o并没有覆盖外层作用域outer中的o

    def outer2(): #
    o = 65
    def inner():
        o = 97
        print("inner {}".format(o))
        print(chr(o))
    print("outer {}".format(o))
    inner()
>>>
outer 65
inner 97
a

    函数的作用域¶

    一个标识符的可见范围,就是这个标识符的作用域.

    • 全局作用域global

    在整个运行环境中都可以调用的,(最外层)

    内部作用域变量使用global将函数内部的x声明为使用外部的全局作用域中定义的x(外部可调用)

    x = 5
def foo():
    global x
  x += 1

    • 不建议使用global ,请使用参数的形参传参解决

    • 局部作用域;local

    局部作用域中的变量叫本地变量,

    在函数,类等内部可见

    局部变量使用范围不能唱过其所在的局部作用域

    变量名的解析原则LEGB¶

    闭包¶

    • 自由变量:围在 本地作用域中定义的变量.例如定义在内层函数外的外层函数的作用域中的变量
    • 闭包:出现在嵌套函数中,指的是内层函数引用到了外层函数的自由变量,对内层函数来说,就形成了闭包
    • 外函数中定义了一个内函数
    • 内函数里运用 了外函数的临时变量
    • 外函数的返回值是内函数的引用

    一般情况下,如果一个函数结束,函数的内部所有东西都会被释放掉,还给内存,局部变量都会消失.但是闭包是一种特殊情况,如果外函数在结束时候发现自己的临时变量将来会在内部函数中用到,就把这个临时变量绑定给内部函数,然后自己再结束.

    可以使用nonlocal 关键字将变量标记在上级的局部作用域中定义,但是**nonlocal**的上一层不能为全局作用域.

    简单来说就是内层函数使用了外层函作用域中的变量.

    函数的销毁¶

    • 全局函数销毁
    • 重新覆盖一个同名函数
    • del语句删除函数对象

    • 程序结束时

    • 局部函数销毁(函数内部)

    • 重新在上级作用域内定义同名函数
    • del语句删除函数对象
    • 上级作用域销毁时

    函数的执行流程¶

    调用函数,函数压栈,参数压栈,创建栈帧(frames),执行,弹出

    举例

    def foo1(b,b1=3):
    print("foo1 called",b,b1)

def foo2(c):
    foo3(c)
    print("foo2 called",c)

def foo3(d):
    print("foo3 called",d)

def main():
    print("main called")
    foo1(100,101)
    foo2(200)
    print("main ending")

main()
------------------------------------------------------------
上面main函数调用执行的流程如下：
全局帧中生成foo1,foo2,foo3,main函数对象
0．main函数调用
1．main中查找内建函数print压栈，将常量字符串压栈，调用函数弹出栈顶．
2．main中全局查找函数foo1压栈，将常量100,101压栈，调用函数foo1，创建栈帧，print函数压栈，字符串和变量b,b1压栈，调用函数，弹出栈顶，返回值．
3．main中全局查找foo2函数压栈，将常量200压栈，调用foo2，创建栈帧，foo3函数压栈，变量c引用压栈，调用foo3，创建栈帧，foo3完成print函数调用后返回，foo2恢复调用，执行print后，返回值．main中foo2调用结束弹出栈顶
4.main函数继续执行print函数调用,弹出栈顶,main函数返回

总结:函数调用—全局查找函数—函数压栈—常量字符串压栈—创建栈帧—执行函数体—返回执行结果–函数出栈

    高阶函数¶

    高阶函数至少满足下面一个条件的函数

    • 接受一个或多个函数作为参数
    • 输出一个函数

    内建高阶函数

    • sorted(iterable[,key][,reverse]) 排序

    返回一个新列表,对一个可迭代对象所有元素排序,排序规则为key定义函数,reverse表示是否反转,默认升序

    list.sort()就地修改,sorted()返回新列表

    • filter(function,iterable) 过滤

    过滤可迭代对象的元素,返回一个==迭代器,==

    function一个具有一个参数的函数,返回bool

    >>>list(filter(lambda x: x%3==0, [1,9,55,150,-3,78,28,123]))
>>>[9, 150, -3, 78, 123]
    • map(function,*iterables) 映射

    返回 map object 对多个可迭代对象的元素按照指定函数映射,返回一个迭代器

    >>>list(map(lambda x:2*x+1, range(5)))
>>>[1, 3, 5, 7, 9]
>>>dict(map(lambda x: (x%5,x) , range(500)))
>>>{0: 495, 1: 496, 2: 497, 3: 498, 4: 499}

    柯里化Currying¶

    柯里化:指将原来接受两个参数的函数变成新的一个接受一个参数的函数的过程 .新的函数返回一个以原有第二个参数为参数的函数

    z = f(x,y) → z = f(x)(y)

    函数参数返回值数据结构¶

    from typing import List, Tuple, Dict
def add(a:int, string:str, f:float, b:bool) -> Tuple[List, Tuple, Dict, bool]:
    list1 = list(range(a))
    tup = (string, string, string)
    d = {"a":f}
    bl = b
    return list1, tup, d,bl
print(add(5,"hhhh", 2.3, False))
# 结果：([0, 1, 2, 3, 4], ('hhhh', 'hhhh', 'hhhh'), {'a': 2.3}, False)
