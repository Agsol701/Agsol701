    计算机的基本组成¶

    • 冯诺依曼体系

    cpu¶

    控制器¶

    控制整个电脑的运行,程序的执行

    运算器¶

    算数运算,逻辑运算,数据传输

    存储器¶

    主存(内存)¶

    优点:速度快

    缺点:容量小,断电即失

    外存¶

    优点:容量大,永久储存

    缺点:速度慢

    I/O设备¶

    输入设备¶

    键盘,鼠标

    输出设备¶

    耳机,音响

    应用程序的启动¶

    1.ROM(read-only memory)通电启动开机BIOS(basic input/output system)

    2.硬件自检

    3.硬盘检索操作系统

    4.启动操作系统

    补充¶

    • 计算机分32位和64位,位数不同导致 安装速度不同

    多核CPU¶

    多核cpu相当于同时处理多见事情

    总线¶

    连接计算机硬件

    机械硬盘¶

    平均寻道(磁道)时间¶

    5ms

    平均延迟时间(机械手臂寻找数据的时间)¶

    平均值

    7200r/min

    8.3ms

    平均4.15ms

    平均寻找数据时间¶

    5+4.15=9.15ms

    文件¶

    文件是操作系统提供的虚拟概念,用来存储信息

    应用程序¶

    文件组合形成的可以实现某种功能

    操作系统¶

    与硬件交互,把对硬件的操作简化

    计算机的组成¶

    软件(应用程序,操作系统)

    硬件(cpu等)

    操作系统 的启动¶

    计算机内部两个存储器

    一个存储器内有个临时操作系统 (BIOS)

    一个存储器存储了临时操作系统路径(电池供电)
