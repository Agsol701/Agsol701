    列表List¶

    • 使用[]括起来,用,分割的数据结构

    • 列表是可变对象

    • 列表的索引从0开始,
    • 支持负索引,最后一位索引为-1,倒数第二位为-2,一次类推

    创建列表¶

    1.使用[]创建

    In [38]: lst = [1,2,3,4,5]

In [39]: lst
Out[39]: [1, 2, 3, 4, 5]

    2.使用list(iterable) 内建函数,()中为可迭代对象,常用于强制类型转换

    In [40]: a
Out[40]: '0a1a2a3a4a5'

In [41]: l = list(a)

In [42]: l
Out[42]: ['0', 'a', '1', 'a', '2', 'a', '3', 'a', '4', 'a', '5']

In [44]: lst.append(6)

Out[45]: [1, 2, 3, 4, 5, 6]

Out[47]: [0, 1, 2, 3, 4, 5, 6]
    • lst.extend(iterable) 尾部追加,扩展列表

    返回值为None,就地修改

    • *,+两个运算符可以在列表之间使用,生成新列表

    切片¶

    Python中符合序列的有序序列都支持切片（slice），例如列表，字符串，元组。

    用于截取有序序列中的一个片段

    语法:object[start:end:step]

    start:起始索引，从0开始，-1表示结束

    end：结束索引

    step：步长，end-start，步长为正时，从左向右取值。步长为负时，反向取值,默认为1

    注意¶
    1. 切片步长的方向非常重要,步长需要与头尾顺序一致
    2. 切片的结果为左开右闭区间,不包含结束索引，即不包含最后的一位，-1代表列表的最后一个位置索引,
    3. 想切到最后一位 : lst[3:]表示从索引3开始后面的所有元素,

    拷贝¶

    由于列表为可变类型容器,所以存在拷贝,浅拷贝和深拷贝的问题,在编程中尤其需要注意.

    拷贝相关内容为容器类元素问题,主要发生在存在列表的情况下.

    拷贝¶

    普通拷贝即使被拷贝对象引用计数加1,是指针的指向多出一个分支指向引用对象.

    内存地址一致

    In [79]: lst = [1,2,3,4]

In [80]: lst1 = lst

In [81]: lst1.append(5)

In [82]: lst1
Out[82]: [1, 2, 3, 4, 5]

In [83]: lst
Out[83]: [1, 2, 3, 4, 5]
    浅拷贝¶

    也叫影子拷贝shadow copy,只拷贝容器类元素中的第一层元素,容器套容器,只是指针的引用

    套中套的内存地址一致

    In [84]: lst = [1,2,3,[4,5]]

In [85]: import copy

In [86]: lst1 = lst.copy()

In [87]: lst1[0,0]

In [90]: lst1
Out[90]: [0, 2, 3, [4, 5]]

In [91]: lst
Out[91]: [1, 2, 3, [4, 5]]

In [92]: lst1[3][0] = 6

In [93]: lst1
Out[93]: [0, 2, 3, [6, 5]]

In [94]: lst
Out[94]: [1, 2, 3, [6, 5]]
    深拷贝¶

    使用copy模块中的deepcopy方法,将源元素完整复制一份到新的内存地址

    内存地址不一致

    In [95]: import copy

In [96]: lst = [1,2,3,[4,5]]

In [97]: lst1 = copy.deepcopy(lst)

In [98]: lst1[3][0] = 6

In [99]: lst1
Out[99]: [1, 2, 3, [6, 5]]

In [100]: lst
Out[100]: [1, 2, 3, [4, 5]]

    元组tuple¶

    • 使用()括起来,用,分割的数据结构
    • 元组为不可变对象,即定义后不可改变数据
    • 当元素中含有列表时,此列表依然为地址索引,所以可以更改列表中的值
    • 使用方法与去掉更改值的方法,与列表一致
    • 同样的数据的情况下,元组所占内存空间比列表更小
    注意:¶
    1. 当想表示一个元素的元组时,需要在元素后加,,而列表不需要
    In [102]: a
Out[102]: [1]

In [103]: type(a)
Out[103]: list

In [104]: b = (1)

In [105]: type(b)
Out[105]: int

In [106]: c = (1,)

In [107]: type(c)
Out[107]: tuple
