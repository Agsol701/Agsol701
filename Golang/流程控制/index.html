    顺序控制¶

    程序从上至下逐行的执行,中间没有任何的判断和跳转.

    分支控制¶

    单分支,双分支,多分支

    其中:

    • 条件表达式可以用小括号括起来,也可以不写,建议不写,因为作者源码就这么写的

    • else不可以换行

    • 表达式部分不可以是赋值语句,一定要是条件语句(即true,或false),

    if b = bool ...语法报错

    • if 后的条件条件表达式也可以使用()括起来

    单分支¶

    if 条件表达式 {
    执行代码块
}

    当条件表达式为true时执行代码块

    也可以在条件之前赋值
if age := 18;age > 18 {
    fmt.Println("be responsible for your behavior")
}

    双分支¶

    if 条件表达式 {
    代码块1
} else {   //else不能放到下一行
    代码块2
}

    var age int 
fmt.Println("age:")
fmt.Scanln(&age)
if age > 18 {
    fmt.Println("1")
}   else {
    fmt.Println("2")
}

    多分支¶

    if 条件表达式1 {
    代码块1
} else if 条件表达式2 {
    代码块2
} 
....
} else {
    代码块3
}

    其中最后一个else可以不存在

    嵌套分支¶

    在一个分支结构中又嵌套了另一个完整的分支结构,里面的分支的结构称之为内层分支,外面的分支称之为外层分支.

    建议最多不超过三层

    switch分支结构¶

    • switch语句用于基于不同条件执行的不同动作,每一个case分支都是唯一的,从上到下逐一测试,直到匹配为止.都执行不到,执行默认分支.

    • case匹配项后面不需要加break

    • 用switch的表达式于case的表达式对比
    • case后的表达式可以有多个,用逗号,隔开

    基本语法

    switch 表达式 {
case 表达式1,表达式2,...:  //表示匹配到表达式1或者2都已执行
    语句块1
case 表达式3,表达式4,...:
    语句块2
    .
    .
    .
default:
    语句块
}

    练习:

    //按输入转换为星期输出
    var key byte
    fmt.Println("enter a byte:")
    fmt.Scanf("%c",&key)

    switch key {
        case 'a':
            fmt.Println("monday")
        case 'b':
            fmt.Println("tuesday")
        case 'c':
            fmt.Println("wednesday")
        case 'd':
            fmt.Println("thursday")
        case 'e':
            fmt.Println("friday")
        case 'f':
            fmt.Println("saturday")
        case 'g':
            fmt.Println("sunday")
        default:
            fmt.Println("cnm")
    }

    细节讨论

    1. case/switch后是一个表达式.(常量值,变量,一个有返回值的函数都可以)

    2. case后的各个表达式的值的数据类型,必须和switch的表达式数据类型一致.

    3. case后面可以带多个表达式,使用逗号间隔.case 表达式1,表达式2....

    switch n1 {
case n2,10,5:  // 只要匹配到任意一个符合的就会执行下面的语句
    fmt.Println("123")

    1. case后面的表达式如果是常量值(字面常量),则要求不能重复.但是如果是值相同的变量,golang无法检测出来,可以骗过编译器.

    2. case后面不需要带break,程序匹配到一个case后就会执行对应的代码块,然后退出switch,如果都执行不到,执行default

    3. default语句不是必须的.

    4. switch后也可以不带表达式,类似if--else分支来使用.

    5. switch后也可以直接声明变量.分号结束,不推荐使用.

     switch grade := 90;{
 case grade > 90:
     fmt.Println("great")
 case grade > 70:
     fmt.Println("ok")
 }
    1. switch 穿透-fallthrough,在case语句块后加fallthrough,则会继续执行下一个case,无论下一个case是否满足,也叫switch穿透.
    var num int = 10
switch num {
    case 10:
    fmt.Println("1")
    fallthrough
    case 20:
    fmt.Println("2")
    fallthrough // 默认穿透一层
    case 30:
    fmt.Println("3")
}
    1. Type Switch:switch语句还可以被用于type_switch来判断某个interface变量中实际指向的变量类型.

    switch于if的比较¶

    1. 判断==具体数值不多==,且符合整数,浮点数,字符,字符串这几种类型,条件比较少,使用switch,简洁高效
    2. ==对区间的判断==和结果为bool类型的判断,使用if,if的使用范围更广.

    循环控制¶

    for循环控制¶

    基本语法¶
    for 循环变量初始化;循环条件;循环变量迭代 {
    循环语句
}
----------------------
for 循环判断条件 {
    循环执行语句 //变量声明初始化和变量迭代可以写到其他位置
}
----------------------
for {  // 等价于for;;{} 是一个无线循环,通常配合break使用
   循环执行语句
}   

    for i:=1; i<10;i++ {
    fmt.Println("i")
}
fmt.Println(i)//报错,外部无法使用undefined i
--------------------------------------------
for i := 0; i < 20; i += 2 {  //也可以自己定义step
    fmt.Println(i)
}
    注意事项¶

    1. for后语句顺序不可变

    2. for循环中声明的变量的作用域只在for循环中.外部无法使用

    3. 循环条件一定是个bool值

    4. 变量声明初始化和变量迭代可以写到其他位置

    j := 1
for j < 10 {
    fmt.Println(j)
    j++
}

    1. Golang提供for-range的方式.可以方便遍历字符串和数组,

    2. for后面的条件执行顺序,i:=1 →i<10→循环体 → i++ → i<10 .....

    3. for之后,{}之前的语句不能用()括起来

    4. for后的语句分三部分,每部分都可省略,根据实际需求而定.

    字符串遍历¶
    方法1传统方式,¶
    var str string = "hello,word"
for i :=0;i<len(str);i++{
    fmt.Printf("%c\n",str[i])
}

    注意:此方法是按照字节来遍历的,汉字会出现乱码,

    解决,需要将 str转成 []切片 来解决

    var str string = "一往无前虎山行"
str2 := []rune(str)
for i := 0;i< len(str2); i++ {
    fmt.Printf("%c\n",str2[i])
}
    方法2 for...range方法(键值循环)¶

    按照字符的方式遍历

    如果不想要索引值,可以使用_下划线接替表示丢弃变量

    var str string = "hello,word上海"
for index,val := range str {
    fmt.Printf("%d,%c\n",index,val)
}

    while¶

    先判断再做

    Go中没有whle和do,,,while语法,可以通过for循环来实现其使用效果.

    1. for循环实现while的效果
    for {
    if 循环条件表达式 {
        break // 跳出for循环
    }
    循环语句
    循环变量迭代
}

    说明:使用死循环for,在内部进行迭代和循环语句,用if判断来控制死循环的结束.

    1. break来结束for循环
    //输入10句hello world    while实现
var i int = 0
for {
    if  i==10 {
        break
    }
    fmt.Println("hello world")
    i++
}

    do...while¶

    先做再判断

    1. for 实现do...while
    for {
    循环语句
    循环变量迭代
    if 循环条件 {
        break
    }
}
    1. 说明:

    先循环,再判断,因此至少执行一次

    当循环成立后,执行break,跳出for循环,结束循环

    //输入10句hello world    do...while实现
var i int = 0
for {
    fmt.Println("hello world")
    i++
    if  i==10 {
        break
    }
}

    循环嵌套¶

    1. 将一个循环放到另一个循环体内,就形成了嵌套循环,外层的for称为外层循环,内层的for称为内层循环.
    2. 建议不超过3层
    3. 设外层循环m次,内层循环n次,则内层循环体实际上需要执行m*n= mn次
    4. 本质:将内层循环当成外层循环的循环体
    //打印输入数字的空心金子塔
var totalLevel int 
fmt.Print("enter a digit:")
fmt.Scanln(&totalLevel)
for i := 1;i<=totalLevel;i++ {
    //打印*前先打印空格
    for k:= 1; k <= totalLevel - i; k++ {
        fmt.Print(" ")
    }
    //j表示每层打印多少*
    for j :=1; j<= 2 *i - 1; j++ {
        if j ==1 || j == 2 * i -1 || i == totalLevel {
            fmt.Print("*")
        } else {
            fmt.Print(" ")
        }
    }
    fmt.Println()
}

    先易后难,先死后活

    //九九乘法表
for i := 1; i <=9; i++ {
    for j := 1; j<=i; j++ {
        fmt.Printf("%v * %v = %v\t",j,i,j*i)
    }
    fmt.Println()
}

for i in range(9):
    for j in range(i):
        print(f"{j+1} * {i} = {(j+1)*i}",end='\t')

    break¶

    终止循环

    break语句在多层嵌套的语句块中时,可以通过标签指明要终止的是哪一层语句块

    label1:{
    label2:{
        label3:{
            break  label2
        } //a
    } //b
} //c

    不写,默认终止本层循环,即break后到a,如加label2,则跳到b,label1就跳到c

    //例子
label1: //设置一个标签
for i := 0;i < 4; i++ {

    for j := 0; j < 10; j++ {
        if j == 2 {
            break label1
        }
        fmt.Print("j=",j)
    }
    fmt.Print("i=",i)
}

    continue¶

    跳出本次循环,开始下一次循环

    continue语句出现在多层嵌套的循环语句中时,可以通过标签指明要跳过的时哪一层循环,规则同break

    跳转控制语句goto¶

    汇编语言中使用比较多,高级语言用的比较少.

    1. Go语言的goto语句可以无条件的转移到程序中指定的行.
    2. goto语句通常与条件语句配合使用,可以用来实现条件转移,跳出循环体等功能.
    3. 在Go程序设计中,不推荐使用goto语句,避免造成程序的混乱,使理解和调试产生困难..
    goto label
..
label:statement

    跳出控制语句return¶

    跳出所在的函数或方法

    如果return时在普通的函数,则表示跳出该函数,即不再执行return后面的代码.

    如果return时在main函数中,终止main主函数,就是终止程序
