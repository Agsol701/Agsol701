    变量¶

    变量是程序的基本组成单位

    变量是内存中一个数据存储空间的表示，通过变量名可以访问到变量值。

    变量使用的基本步骤¶

    1. 声明变量（定义变量）
    2. 赋值
    3. 使用
    package main
import "fmt"
func main(){
    var i int = 10
    fmt.Println("i=",i)
}
>>>i=10

    变量的声明，初始化和赋值¶

    声明变量¶

    基本语法：var 变量名 数据类型

    var a int 声明了一个整型变量a

    初始化变量¶

    在声明的时候就给一个值 

    var a int = 5  初始化一个整型变量使其值为5

    如果声明时就直接赋值，可省略数据类型

    ​ var b = 400

    给变量赋值¶

    先声明了变量var num int 此时使用默认值为0

    然后赋值 num= 100

    变量的初始化格式¶

    变量标准定义初始化格式¶
    var a int = 10
    编译器推到类型的格式¶
    var b = 10
    短变量声明并初始化¶
    c := 10

    变量使用注意事项¶

    golang中变量要先声明再使用

    1. 变量表示内存中的一个存储区域

    2. 该区域有自己的名称（变量名）和类型（数据类型）

    3. Golang变量声明的三种方式：

    4. 指定变量类型，声明后若不赋值，使用默认值

    5. 根据值自动判定变量类型（类型推导）

    6. 省略var 注意 ：=左侧变量不应该是已经声明过的，否则会导致编译错误

      name := "tom"
等价于
var name string
name = "tom"
等价于
var name = "tom"

    7. 多变量声明

    在变成中，有事需要一次性声明多个变量，golang也提供这样的语法

    1. 方式1 多个同一数据类型

      var n1,n2,n3 int

    2. 方式2 多个不同数据类型

      var n1,name,n3 = 100,"tom",888

      1. 方式3 类型推导

      n1,name,n3 ：= 100,"tom",888 等价于方式2

    注意：在go中，函数外声明的变量就是全局变量，

    全局变量的声明

    var n1 = 100
var name = "tom"
等价于
var(
 n1 = 100
    name = "tom"
)

    1. 在该区域的数据值可以在统一类型范围内不断变化，即同数据类型可覆盖赋值

    2. 变量在同一个作用域内不能重名

    3. 变量三要素

    变量=变量名 + 值+数据类型

    1. golang的变量如果没有赋值，编译器会使用默认值，int，float默认0，string默认为空串
    注意：¶

    golang中定义变量，会先根据开辟内存空间，确定数据类型，给定默认值，在赋值的时候再将默认值更改为被赋值的值。

    golang中:=表示，定义赋值

    程序中+号的使用¶

    1. 当左右两边都是数值类型时，做加法运算

    2. 当左右两边都是字符串类型时，做字符串拼接

    3. 数字和字符串不能使用+号

    (mismatched types string and int)

    常量¶

    • 常量使用const修改

    • 常量在定义的时候,必须初始化

    • 常量不能修改

    • 常量只能修饰bool,数值类型(in,float系列,),string类型

    • 语法:const identifier [type] = value

    • go中没有规定常量一定要全大写,但是依然通过首字母大小写来控制常量的访问范围.

    • 举例:

    const name = "tom" 对

    const b = num /3 num为变量,则报错

    简洁的写法¶

    const (
    a = 1
    b =2
)

    专业的写法¶

    const (
    a = iota //表示a赋值为0
    b       //b在a的基础上+1
    c       //c在b的基础上+1
)
fmt.Println(a,b,c) //0,1,2
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
const (
    a = iota //表示a赋值为0
    b = iota //b在a的基础上+1
    c,d = iota,iota //c,d在b的基础上+1
)
fmt.Println(a,b,c,d) //0,1,2,2

    iota只在本const()的括号中起效,出了括号的iota重新从0开始

    无类型常量¶

    Go语言的常量的不同之处.很多常量没有一个明确的基础类型.编译器为这些没有明确的基础类型的数字常量提供比基础类型更高精度的算数运算.可以认为至少有256bit的运算精度.这里有六种未明确类型的常量类型,分别时无类型的布尔型,无类型的整数,无类型的浮点数,无类型的字符,无类型的复数,无类型的字符串.

    通过延迟明确常量的具体类型,无类型的常量不仅可以提供更高的运算精度,而且可以直接用于更多的表达式而不需要显式的类型转换.
