    函数基础¶

    目的¶

    减少代码冗余

    方便代码的维护

    解决代码复用性的问题

    概念¶

    为了完成某以功能的程序指令(语句)的合集,称为函数.

    分类¶

    自定义函数,系统函数(Go编程手册)

    基本语法¶

    func 函数名 (形参列表) (返回值列表) {
    执行语句
    return 返回值列表
}
    1. 函数名的命名与标识符一致．且在同一个包内，函数名不能重复
    2. 形参列表:表示函数的输入，由参数变量和参数变量的类型组成，多个参数之间用,分割
    3. 函数中的语句:表示为了实现某一功能代码块
    4. 函数的参数，返回值也可以没有

    函数的调用¶

    定义函数之后,可以通过 函数名() 的方法调用.

    调用函数的时候也可以不接收其返回值.

    注意:函数的返回值可以不接收,但是不可以接收部分,(要么全接收,要不不接收)

    内存流程¶
    1. 在调用一个函数时,会给该函数分配一个新的空间,编译器会通过自身处理让这个新的空间和其他空间的栈的空间区分开
    2. 每个函数对应的栈中,数据空间是独立的,不会混淆
    3. 当一个函数结束调用(执行完毕)后,程序会销毁这个函数对应的栈空间.

    类型简写¶

    函数的从参数中如果相邻变量的类型相同,则可以省略类型

    func intSum(x,y int) int {  //当x,y类型相同时,可在最后只写一个类型.
    return x+y
}

    sum := 0

    参数的传递¶
    分类¶
    1. 值传递:基本数据类型,

    基本数据类型(int系列,float系列,bool,string),数组,结构体

    值传递就是在传递参数的时候进行了值拷贝.

    1. 引用传递:传递的是地址.

    指针,slice切片,map,管道channel,interface等默认引用类型,

    区别:¶

    无论值传递还是引用传递,传递给函数的都是变量的副本,

    值传递的是值的拷贝,引用传递传递的地址的拷贝

    一般来说,地址拷贝效率高,因为数据量小

    而值拷贝决定于数据的大小,数据越大,效率越低.

    返回值¶

    函数可以没有返回值

    func test(n1 int) {
    n1 = n1 +1
    fmt.Println(n1)
}

    Go语言中通过return关键字向外输出返回值.

    注意:¶

    函数中如果没定义返回值,函数中的return只是终止函数,不返回任何值.

    多返回值¶

    Go语言中支持多返回值,函数如果有多个返回值的时候,必须用()将所有返回值包裹起来.

    func calc(x,y int) (int,int) {
    sum := x+y
    sbu := x-y
    return sum,sub
}

    不想用的返回值可以使用_接收,丢弃变量

    命名返回值¶

    函数定义时可以给返回值命名,并在函数体中直接使用这些变量,最后直接return关键字返回

    func calc(x,y int) (sum, sub int) {
    sum := x+y
    sbu := x-y
    return 
}

    这时就是使用了返回值的默认值,return关键字会去函数定义中找返回值并返回.

    函数进阶¶

    变量作用域¶

    全局变量¶

    函数外部定义/声明的变量叫全局变量,作用域在整个包都有效,如果其首字母大写,则作用域在整个程序都有效.

    全局变量时定义在函数外部的变量,它在程序整个运行周期内部都有效,在函数中可以访问到全局变量,

    package main
import "fmt"
var num int = 10
func main (){
    fmt.Println(num)
}
>>>10
    局部变量¶

    在函数内部声明/定义的变量叫局部变量,作用域仅限于函数内部.

    1. 函数内定义的变量无法在该函数外使用.
    2. 如果局部变量和全局变量重名,优先访问局部变量.for if switch与函数同理

    函数类型与变量¶

    定义函数类型¶

    可以使用type关键字来定义一个函数类型

    type calculation func(int, int) int
//定义了一个calculation类型,这是一种函数类型,这种函数接收两个int类型的参数并且返回一个int类型的函数

    满足这个条件的函数都是calculation类型的函数,

    函数类型变量¶
    type calculation func(int,int) int //定义了一个calculation函数类型,输入两个int返回一个int

func add(x,y int) int {
    return x+y
}
func main (){
    var c calculation = add  //声明一个calculation类型的变量c并把add赋值给c
    res := c(1,3)  //像调用add一样调用c
    fmt.Println(res) //4
    fmt.Printf("%T\n",c) //main.calculation
}
>>>
4
main.calculation

    高阶函数¶

    高阶函数即函数作为参数或者函数作为返回值.

    函数作为参数¶
    func add(x,y int) int {
    return x+y
}
//func后的4个int,分别为add的两个参数,一个返回值和calc的返回值
//calc共有三个参数,两个int和一个func
func calc(x,y int,op func(int,int)int)int {

    func add(x,y int) int {
    return x+y
}

func do() (func(int,int)int) {
    return add
}

func main(){
    res := do()(20,30)
    fmt.Println(res)
}

    递归调用¶

    一个函数在函数体内又调用了函数本身,就称为递归调用.

    例1:

    package main
import "fmt"
func test(n int) {
    if n> 2{
        n--
        test(n)
    }
    fmt.Println(n)
}

func main() {
    test(4)
}
>>>
2
2
3

    递归返回的时候回到了上一个if,出栈的时候执行了Print

    进栈的时候,Print在栈中等待执行,出栈的时候执行

    例2

    package main
import "fmt"
func test(n int) {
    if n> 2{
        n--
        test(n)
    } else {
        fmt.Println(n)
    }
}

func main() {
    test(4)
}
>>>
2

    出栈的时候进不到else里面,所以只打印一个2

    原则:

    1. 执行一个函数时,就创建一个新的受保护的独立空间(新函数栈)
    2. 函数的局部变量时独立的,不会相互影响
    3. 递归必须向退出递归的条件逼近,否则就是无限递归
    4. 当一个函数执行完毕,或者遇到return,就会返回,遵守谁调用,就将结果返回给谁.
    5. 当函数执行完毕或者返回时,函数本身也会被销毁.

    练习:

    //斐波那契数
func fib (n int) (int){
        if (n == 1 || n ==2) {
            return 1
        } else {
            return fib(n-1) + fib(n-2)
    }
}

func main (){
    fmt.Println(fib(5))
}
>>>5
//f(1)=3,f(n)=2*f(n-1)+1,求f(n)

    if n==1 {
        return 3
    } else {
        return 2*f(n-1)+1
    }
}
func main() {
    fmt.Println(f(5))
}
>>>63

    init函数¶

    每一个源文件都可以包含一个init函数,该函数会在main函数执行前,被Go运行框架调用,也就是说,init会在main函数前被调用.

    通常在init函数中完成函数的初始化工作.

    注意:¶
    • 如果一个文件同时包含全局变量定义,init函数和main函数,则执行的流程是

    变量定义 → init函数 → main函数

    面试题:

    如果main.go和utils.go都含有变量定义,init函数时,执行的流程会是怎样的?

    从最外面的被导入包,依次向main函数靠拢.每个源文件都是先全局变量定义,然后init函数,然后下一层.

    匿名函数¶

    如果,一个函数定义了只希望使用一次,可以考虑使用匿名函数.但是匿名函数也可以实现多次调用.

    多用于实现回调函数和闭包.即在函数内部使用匿名函数.

    匿名函数只能在函数内使用,

    定义方式¶
    func(参数)(返回值){
    函数体
}
    使用方法¶
    1. 定义后直接加()调用
    //自执行函数：匿名函数定义完加()直接执行
func(x, y int) {
    fmt.Println(x + y)
}(10, 20)
}

    这种方式匿名函数只能调用一次.

    1. 将匿名函数赋值给一个变量(函数变量),再通过该变量来调用匿名函数

    ```go func main() { // 将匿名函数保存到变量 add := func(x, y int) { fmt.Println(x + y) } add(10, 20) // 通过变量调用匿名函数 fmt.Printf("%T", add) }

    30 func(int, int) ```

    虽然add可以被反复调用,但是add不是函数名,可以通过这种方法在mian函数内定义函数

    这里的add是func(int, int)类型

    全局匿名函数¶

    如果将匿名函数赋值给一个全局变量,那么这个匿名函数,就称为一个全局匿名函数,可以在全局被调用.

    var (
    Fun = func(n1,n2 int) int {  // 变量名记得大写
        return n1 + n2
    }
)
func main (){
    res := Fun(3,6)
    fmt.Println(res)
}

    闭包¶

    闭包函数,就是多了一个函数传参的方式.

    闭包就是一个函数和与其相关的引用环境组合的一个整体.

    闭包 = 函数 + 引用环境

    会形成局部变量的驻留

    //累加器
func add () func(int) int {
    var x int  
    return func (y int) int {
        x += y
        return x
    } 
}

func main () {
   var f = add() //返回一个闭包
    fmt.Println(f(10))
    fmt.Println(f(20))

    f1 := add()  //返回一个闭包
    fmt.Println(f1(10))
    fmt.Println(f1(20))
}

    说明:

    1. add时一个函数,返回的数据类型时func (int) int

    2. 闭包说明:这个部分就是一个闭包

    var x int  
return func (y int) int {
    x += y
    return x

    返回的是一个匿名函数func (y int) int,但是匿名函数引用到了函数外的变量x,因此这个匿名函数于x就形成一个整体,构成闭包.

    1. 理解:闭包里类(的单例模式),匿名函数是方法,被引用变量是属性.匿名函数与被引用的变量就构成了一个闭包.

    2. 当反复调用main里面的f时,因为x只初始化一次,因此每调用一次就是实现了累加,而不会重新初始化变量x.

    3. 分析时要注意返回值里面的匿名函数引用到了哪些变量,因为函数和它引用的变量共同构成了闭包.

    实践

    //判断字符串是否有指定后缀
import "strings"
func makeSuffix (suffix string ) func (string) string {
    return func (name string) string {
        if !strings.HasSuffix(name,suffix)  {
            return name + suffix
        }
        return name
    }
}

func main (){
    f := makeSuffix(".jpg")
    fmt.Println(f("aaa.jpg"))
}

    说明:

    返回的函数与makeSuffix中的suffix变量组成一个闭包,因为返回函数用到了suffix这个变量

    闭包的好处,可以不用重复定义,只定义一次就可以了.

    defer的使用¶

    延时机制,在本函数体内执行到defer时,将defer后的语句(1,2,3)压栈,等其他语句执行完毕,依次(3,2,1)出栈.

    延迟处理在defer归属的函数即将返回时，将延迟处理的语句按defer定义的逆序进行执行，也就是说，先被defer的语句最后被执行，最后被defer的语句，最先被执行。

    func main() {
    fmt.Println("start")
    defer fmt.Println(1)
    defer fmt.Println(2)
    defer fmt.Println(3)
    fmt.Println("end")
}
>>>
start
end
3
2
1
    用途:¶

    由于defer语句的延迟调用的特性,所以defer语句可以非常方便的处理资源释放的问题.

    比如:

    创建资源(数据库连接,文件句柄,锁等),资源清理,文件关闭,解锁即记录时间等.

    练习:

    func sum(n1,n2 int) int {
    defer fmt.Println("1",n1)
    defer fmt.Println("2",n2)

    res := n1 +n2
    fmt.Println("3",res)
    return res
}
func main () {
    res := sum(10,20)
    res += 1
    fmt.Println("res",res)
}
>>>
3 30
2 20
1 10
res 31
//在defer将语句放入栈中时,也会将相关的值拷贝同时入栈.
func sum(n1,n2 int) int {
    defer fmt.Println("1",n1)
    defer fmt.Println("2",n2)
    n1 ++
    n2 ++
    res := n1 +n2
    fmt.Println("3",res)
    return res
}
func main () {
    res := sum(10,20)
    res += 1
    fmt.Println("res",res)
}
>>>
3 32
2 20
1 10
res 33

    说明:在defer将语句放入栈中时,也会将相关的值拷贝同时入栈.

    注意事项和细节¶

    1. 函数的形参列表可以是多个或0个,返回值列表也可以是多个或0个.

    2. 形参列表和返回值列表的数据类型可以是值类型或引用类型.

    3. 函数的命名遵循标识符命名规范.首字母不可以是数字,首字母大写可以被其他包使用,类似public,首字母小写,只能被本包使用,类似private.

    4. 函数中的变量是局部的,函数外不能使用.

    5. 基本数据类型和数组默认都是值传递,即进行值拷贝.

    值类型,在函数内修改,不影响原来的值.

    指针类型,传递的是指针的指针.源地址的指针不变,复制一个新指针指向源指针,所以可以改变源值.

    https://www.jianshu.com/p/172a52ded985

    1. 如果希望函数内的变量可以修改函数外的变量(map,切片本身就是引用,所以这里默认说的是值传递方式的变量),可以传入变量的地址\&,函数内以指针的方式操作变量.从效果上看类似引用.

    2. Go函数不支持函数重载.会被认为是重复定义.

    函数重载:函数名相同,参数类型或个数不同的两个函数.

    1. 在Go中,函数也是一种数据类型,可以赋值给一个变量,则该变量就是一个函数类型的变量.可以通过该变量对函数进行调用.
    func getSum(n1 ,n2 int) int {
 return n1 + n2
}
func main (){
 a := getSum
 fmt.Printf("type of a is %T,type of getSum is %T",a,getSum)
}
>>>
type of a is func(int, int) int,type of getSum is func(int, int) int

    1. 函数也是一种数据类型,所以在Go中可以使用函数作为形参,并且调用.

    2. 支持对函数返回值命名

      func cal(n1, n2 int) (sum,sub int) {
    sum = n1 +n2
    sub = n1 - n2
    return
}

      优点:返回可以不看顺序,按给定的变量名返回.

    3. go支持可变长参数

      func sum (n1 int,args...int) int {
    sum := n1
    //遍历args求值
    for i:= 0;i<len(args);i++{
    sum += args[i]  
    }
    return sum
}
//可支持输入至少一个值,求和

    4. 函数的执行语句,都要放入函数体之内.比如:函数外的赋值语句会报错.

    5. 函数的类型是func(形参类型)返回值类型,形参类型,返回值类型也是函数类型的一部分,函数名,形参名,返回值名都不是类型的一部分

    6. 在函数参数中建议使用切片,不要使用数组.

    内置函数介绍¶

    内置函数 介绍
    close 主要用来关闭channel
    len 用来求长度，比如string、array、slice、map、channel
    new 用来分配内存，主要用来分配值类型，比如int、struct。返回的是指针
    make 用来分配内存，主要用来分配引用类型，比如chan、map、slice
    append 用来追加元素到数组、slice中
