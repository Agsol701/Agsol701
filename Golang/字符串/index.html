    字符串类型string¶

    细节¶

    1. 字符串就是一串固定长度的字符连接起来的字符序列.go的字符串时由单个字节连接起来的.go语言的字符串的字节使用utf-8编码表示Unicode文本

    2. 字符串式不可变类型,一旦赋值了,就不能更改了

    var str = "hello"
str[0] = "a"
>>>cannot assign to str[0]

    1. 字符串的两种形式

    2. 双引号,会识别转义字符

    3. 反引号,以字符串的原生形式输出,包括换行和特殊字符,可以实现防止攻击,输出源代码等效果.包括换行.

    4. 字符串的拼接方式

    5. 加号

      var str = "hello " + "world"
fmt.Println(str)
>>>hello world

      注意,当字符串过长需要换行时,要把加号留在上面一行,否则报错

      var num1 int = 99
var num2 float64 = 23.456
var b bool = true
var myChar byte = 'h'
var str string

str = fmt.Sprintf("%d",num1)
fmt.Printf("1 str type %T %v\n",str,str)

str = fmt.Sprintf("%f",num2)
fmt.Printf("2 str type %T %v\n",str,str)

str = fmt.Sprintf("%t",b)
fmt.Printf("3 str type %T %v\n",str,str)

str = fmt.Sprintf("%c",myChar)
fmt.Printf("4 str type %T %v\n",str,str)

>>>1 str type string 99
>>>2 str type string 23.456000
>>>3 str type string true
>>>4 str type string h

    string转基本数据类型转¶

    使用strconv包的函数¶

    func ParseBool(str string) (value bool, err error) 之类

    会返回两个值,第一个是数据类型值,第二个是err error,所以要用_忽略变量接收

    注意事项¶

    1. 将string转为其他数据类型的时候,要确保string类型可以转成有效数据.比如,可以把'123'转成一个整数,但不能把"hello"转成一个整数,当转换失败的时候,Golang直接将其转换成 值为默认值0 类型为参数中设置的类型 的一个值.

    发散一下,数据类型之间的转换,若转换失败,都会变成该数据类型的默认值.

    单引号会将单引号中的数据转换为ascii码保存,

    字符串系统函数¶

    常用内置函数¶

    1. func len

    func len(v Type) int

    基于v的类型,返回v的长度,

    数组：v 中元素的数量。
数组指针：*v 中元素的数量（即使 v 为 nil）。
切片或映射：v 中元素的数量；若 v 为 nil，len(v) 即为零。
字符串：v 中字节的数量。
通道：通道缓存中队列（未读取）元素的数量；若 v 为 nil，len(v) 即为零。

    package main
import(
 "fmt"
)
func main()  {
 str1 := "hello"
 str2 := "你好"
 str3 := "123"
 fmt.Println(len(str1),len(str2),len(str3))
}
>>>
5 6 3

    4. 将指定的字串替换成另一个字串

      strings.Replace("go go hello","go","Python",n),n表示替换几个,如果 n=-1 表示全部替换

      当n大于实际存在的数量,替换全部,n必须存在

       func main()  {
    a := strings.Replace("go go hello","go","Python",3)
    fmt.Println(a)
 }
 >>>
 Python Python hello

       func main()  {

 }
