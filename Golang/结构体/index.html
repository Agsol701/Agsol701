    结构体¶

    Go语言中没有“类”的概念，也不支持“类”的继承等面向对象的概念。Go语言中通过结构体的内嵌再配合接口比面向对象具有更高的扩展性和灵活性。

    声明¶

    使用type和struct关键字来定义结构体.

    type 类型名 struct {
    字段名 字段类型
    字段名 字段类型
    ...
}

    注意:

    • 类型名:标识自定义结构体的名称,在同一个包内不可重复(无法重构)
    • 字段名:标识结构体字段的名字,结构体中字段名必须唯一.
    • 字段类型:表示结构体的具体类型.
    //比如定义一个人结构体
type person struct {
    name,city string  // 同类型的字段也可以写在同一行
    gender string
    age int8
}

    结构体的字段\属性¶

    1. 字段声明语法同变量: 字段名 字段类型

    2. 字段类型可以为:基本类型,数组或者引用类型

    3. 在创建一个结构体变量后,如果没有赋值,都会对应给该数据结构的默认值(零值),其中指针,slice,和map的零值是nil,需要先make才可以使用.

    type Person struct {
 Name  string
 Age   int
 Score [5]float64
 ptr   *int
 slice []int
 map1  map[string]string
}

func main() {
 var p1 Person
 fmt.Println(p1)
 if p1.ptr == nil {
     fmt.Println("ptr")
 }
 if p1.slice == nil {
     fmt.Println("slice")
 }
 if p1.map1 == nil {
     fmt.Println("map1")
 }
 p1.slice = make([]int, 3)
 p1.slice[0] = 23
 p1.map1 = make(map[string]string)
 p1.map1["key1"] = "value1"
 fmt.Println(p1)
}
>>>
{ 0 [0 0 0 0 0] <nil> [] map[]}
ptr
slice
map1
{ 0 [0 0 0 0 0] <nil> [23 0 0] map[key1:value1]}
    1. 不同结构体变量字段是独立的,互补影响,一个结构体变量字段的更改,不影响另一个对象,即结构体是值类型.

    结构体初始化¶

    没有初始化的结构体,其成员变量都是对应其类型的零值.

    type person struct {
    name string
    city string
    age  int8
}

func main() {
    var p1 person
    fmt.Printf("p1:%#v", p1)
}
>>>
p1:main.person{name:"", city:"", age:0}

    注意:指针,slice和map的零值都是nil,即还没有分派空间,没法用

    使用键值对初始化¶

    键对应结构体的字段,值对应该字段的初始值,

    若使用这种方法,每个字段后都必须加,

    type person struct {
    name string
    city string
    age  int8
}

func main() {
    p1 := person{
        name: "james",
        city: "london",
        age:  30,
    }
    fmt.Printf("p1=%#v", p1)
}
>>>
p1=main.person{name:"james", city:"london", age:30}

    也可以直接生成指针类型的结构体

    type person struct {
    name string
    city string
    age  int8
}

func main() {
    p1 := &person{
        name: "james",
        city: "london",
        age:  30,
    }
    fmt.Printf("p1=%#v", p1)
}
>>>
p1=&main.person{name:"james", city:"london", age:30}

    使用值的列表初始化¶

    初始化结构体的时候可以简写,也就是初始化的时候不写键,直接写值

    type person struct {

    city string
    age  int8
}

func main() {
    p1 := person{
        "james",
        "london",
        30,
    }
    fmt.Printf("p1=%#v", p1)
}
>>>
p1=main.person{name:"james", city:"london", age:30}

}

func main()  {
    var stu1 = &Stu{"agsol",19}
    stu2 := &Stu{"tom",20}
    fmt.Println(stu1,stu2)
    fmt.Println(*stu1,*stu2)
}
>>>
&{agsol 19} &{tom 20}

    使用键值对方式初始化的时候,要注意:

    1. 必须初始化结构体的所有字段
    2. 初始值的填充顺序必须与字段在结构体中声明顺序一致.
    3. 该方式不可以与键值对初始化方式混用.

    结构体的实例化¶

    只有当结构体实例化时,才会真正的分配内存.也就是说必须实例化后才能使用i结构体的字段.

    结构体本身也是一种数据类型,可以像声明内置类型一样使用var关键字声明结构体类型.

    var 结构体实例 结构体类型

    基本实例化¶

    type person struct{
    name,city string
    gender string
    age int8
}
func main()  {
    var p1 person
    p1.name = "agsol"
    p1.city = "shagnhai"
    p1.gender = "male"
    fmt.Println(p1)
}
>>>
{agsol shagnhai male 0}
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
func main()  {
    var p1 person
    p1.name = "agsol"
    p1.city = "shagnhai"
    p1.age =1
    fmt.Println(p1)
    fmt.Println(p1.name)
}
>>>
{agsol shagnhai  1}
agsol

    初始化后会给默认值,int8默认值0

    也可以通过.来访问结构体的字段.

    匿名实例化¶

    在定义一些临时数据结构时还可以使用匿名实例化.

    func main() {
    var user struct {
        Name string
        Age  int
    }
    user.Name = "agsol"
    user.Age = 3
    fmt.Println(user)
}
>>>
{agsol 3}

    创建指针类型结构体¶

    可以使用new关键字对结构体进行实例化,得到的是结构体的指针.

    type person struct {
    name, city string
    gender     string
    age        int8
}
func main() {
    var p1 = new(person)
    fmt.Printf("type:%T\n", p1)
    fmt.Printf("value:%#v\n", p1)
    p1.name = "tom"
    p1.age = 18
    fmt.Println(*p1)
    fmt.Println(p1)
}
>>>
type:*main.person
value:&main.person{name:"", city:"", gender:"", age:0}
{tom   18}
&{tom   18}

    此时p1是一个指针对象,取值可以使用*,printf中可是使用%#v来进行键值对输出

    取址对结构体进行实例化¶

    使用&对结构体进行取地址操作相当于对该结构体类型进行了一次new的实例化操作.

    type person struct {
    name string
    city string
    age  int8
}
func main() {
    p1 := &person{}
    fmt.Printf("%T\n", p1)
    fmt.Printf("p1=%#v\n", p1)
    fmt.Printf("p1=%#v\n", *p1)
    p1.name = "jerry" //等价于(*p1).name = "jerry"
    p1.city = "cy"
    p1.age = 20
    fmt.Printf("p1=%#v\n", p1)
    fmt.Printf("p1=%#v\n", *p1)
}
>>>
*main.person
p1=&main.person{name:"", city:"", age:0}
p1=main.person{name:"", city:"", age:0}
p1=&main.person{name:"jerry", city:"cy", age:20}
p1=main.person{name:"jerry", city:"cy", age:20}

    new得到的是结构体指针,其中p1.name="jerry"在底层是(*p1).name="jerrry",是Go语言帮忙实现的语法糖

    结构体的内存布局¶

    结构体占用一块连续的内存

    type test struct {
    a int8
    b int8
    c int8
    d int8
}
func main()  {
    n := test{
        1,2,3,4,
    }
    fmt.Printf("n.a:%p\n",&n.a)
    fmt.Printf("n.b:%p\n",&n.b)
    fmt.Printf("n.c:%p\n",&n.c)
    fmt.Printf("n.d:%p\n",&n.d)
}
>>>
n.a:0xc0000100a0
n.b:0xc0000100a1
n.c:0xc0000100a2
n.d:0xc0000100a3

    这是go语言中的内存对齐机制https://segmentfault.com/a/1190000017527311?utm_campaign=studygolang.com&utm_medium=studygolang.com&utm_source=studygolang.com

    结构体字段的可见性¶

    结构体中字段大写开头表示可公开访问，小写表示私有（仅在定义当前结构体的包中可访问）。

    结构体相等性¶

    结构体是值类型。

    如果它的每一个字段都是可比较的，则该结构体也是可比较的。

    如果两个结构体变量的对应字段相等，则这两个变量也是相等的.

    如果结构体包含不可比较的字段，则结构体变量也不可比较。

    注意¶

    1. 结构体是值类型.其零值,是属性的零值.

    2. 结构体的自己的字段在内存中是连续的.

    3. 结构体进行type重新定义(相当于取别名),golang认为是新的数据类型,但是相互间可以强转,强转需要两个结构体结构完全一致.

    //2和3的举例
type A struct {
 Num int
}
type B struct {
 Num int
 // Name string
}
func main()  {
 var a A
 var b B
 a = A(b)
 fmt.Println(a,b)
}
    1. struct的每个字段上,可以写上一个tag,该tag可以通过反射机制获取,常见的使用场景就是序列化和反序列化.
    type Monster struct{
 Name string `json:"name"` //`json:"name"`救赎struct tag
 Age int `json:"age"`
 Skill string `json:"skill"`
}
func main()  {
 monster := Monster{"cow", 500, "jiao"}
 //将monster序列化成一个json字符串
 jsonMonster,err := json.Marshal(monster)
 if err != nil {
     fmt.Println("error")
 }
 // fmt.Println(jsonMonster) //返回一个[]byte切片
 fmt.Println(string(jsonMonster))
}
>>>
{"name":"cow","age":500,"skill":"jiao"}

    • 因为使用了json,相当于在json包中使用了本包中的Monster,如果字段名不大写,json就无法引用,为了让前台更舒服的接收,可以使用tag方法让输出值变为tag值,不写tag,默认输出原字段名.

    • tag是结构i提的元信息,可以在运行的时候通过反射的机制读取出来.tag在结构体字段的后方,使用一对反引号包裹起来.

    注意:

    1. 结构体标签由一个或多个键值对组成.键与值使用:隔开,值用""括起来.键值之间使用一个空格分开.
    2. 为结构体编写tag时,必须严格遵守键值对的规则,结构体标签的解析代码的容错能力比较差,一旦格式错误,编译和运行都不会提示错误,但是通过反射又无法正确取值.比如;不要在key和value之间添加空格.

    案例¶

    1) 一个景区根据游人的年龄收取不同价格的门票，比如年龄大于 18，收费 20 元，其它情况门票免 费.

    2) 请编写 Visitor 结构体，根据年龄段决定能够购买的门票价格并输出

    type Vister struct {
    Name string
    Age int
}
func (v *Vister) shouPrice() {
    if v.Age >=90 || v.Age <= 12 {
        fmt.Println("can't in ")
        return
    }
    if v.Age > 18 {
        fmt.Printf("name of vister is %v age is %v price is 20\n",v.Name,v.Age)
    } else {
        fmt.Printf("name of vister is %v age is %v free\n",v.Name,v.Age)
    }
}
func main()  {
    var v Vister
    for {
        fmt.Println("name please:")
        fmt.Scanln(&v.Name)
        if v.Name == "n" {
            fmt.Println("quit")
            break
        }
        fmt.Println("age please:")
        fmt.Scanln(&v.Age)
        v.shouPrice()
    }
}
