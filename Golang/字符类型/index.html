    字符类型(char)¶

    1. golang中没有专门的字符类型,如果要存储单个字符(ascii中),一般使用byte来保存.

    2. golang的字符串,是由==字节==组成的.

    3. 字符常量用单引号括起来的单个字符.

    4. go语言使用utf-8编码,应位字母1个字节,汉字3个字节

    5. go中,字符的本质是一个整数,直接输出,是该字符对应的utf-8编码的码值.

    6. 单个字符类型可以进行运算,相当于一个整数,都对应Unicode编码.
    7. 赋值时可以给某个变量一个数字,然后格式化输出%c,就可以输出该数字对应的unicode字符.
    单引号¶

    单引号在go语言中表示golang中的rune(int32)或byte类型，单引号里面是单个字符，对应的值为该字符的ASCII码值。

    如果使用格式化输出,即可显示原值

    package main
import "fmt"
func main(){
    var i byte = 'a'
    var a byte = '1'
    var b byte = 1
    var n1 = 10 + 'a'
    fmt.Println(i)
    fmt.Println(a)
    fmt.Println(b)
    fmt.Printf("i = %c",i) // %c显示原值  %d输出码值
    fmt.Println("n1 = ", n1)
}
>>>97
>>>49
>>>1
>>>i= a
>>>n1 = 107
    双引号¶

    在go语言中双引号里面可以是单个字符也可以是字符串，双引号里面可以有转义字符，如\n、\r等，对应go语言中的string类型。

    反引号¶

    反引号中的字符表示其原生的意思，在单引号中的内容可以是多行内容，不支持转义。

    本质¶

    1. 字符型存储到计算机中,需要将对应的码值(整数)找出来

    存储:字符→对应码值→二进制→存储

    读取:二进制→码值→字符→读取

    1. 字符和码值的对应关系时通过字符编码表决定的

    2. go语言统一成了utf-8．方便，统一．不易乱码．
