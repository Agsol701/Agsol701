    整数类型¶

    一个字节为9位,8位中第一个位为符号位,1表示符号,所以整数少一个数

    整数类型分类¶

    类型 有无符号 占用空间 表数范围
    int8 有 1字节 -128~127
    int16 有 2字节 -2**15~2**15-1
    int32 有 4字节 -2**31~2**31-1
    int64 有 8字节 -2**63~2**63-1
    uint8 无 1字节 0~255
    uint16 无 2字节 0~2**16-1
    uint32 无 4字节 0~2**32-1
    uint64 无 8字节 0~2**64-1
    int 有 32位系统上占4个字节int32,64位机器上占8个字节,int64
    uint16 无 32位占4个字节,64位占8个字节
    rune 有 跟int32一样 的别名 等价int32,表示一个Unicode码
    byte 无 跟uint8等价0~255 的别名 当要存储字符时选用byte

    8,16,32表示比特位

    注意:

    1. 如int8给了128 抛overflow异常

    2. golang默认整型声明为int型

    3. 在程序中查看某个变量占用的数据类型和字节大小

    import(
    "fmt"
 "unsafe"
)
fmt.Printf("n1 的类型 %T 占用的字节数是", n1, unsafe.Sizeof(n1))

    1. golang程序中整型变量在使用时,遵循保小不保大的原则.即在保证程序正常运行的情况下,尽量使用占用空间小的数据类型.

    2. bit:计算机中最小的存储单位.byte:计算机中基本存储单元.1byte = 8bit

    浮点类型¶

    用浮点类型来表示一个小数.

    小数类型分类¶

    类型|占用空间|表数范围 :-:|:-:|---|--- 单精度float32|1字节|-128~127 双精度float64|1字节|-128~127

    注意:

    1. 浮点数在机器中存放形式:浮点数=符号位+指数位+尾数位

    2. 浮点数都是有符号的

    3. 尾数位部分可能丢失,系统保存时候就记个大概在存储过程中,精度会有丢失

    4. golang浮点型有固定的范围和字段长度,不受OS影响.

    5. golang浮点型默认声明位float64类型.

    6. 5.123e2 = 5.123 * (10**2)

    7. 浮点数也可以显示为整数(看起来像整数)

    var n float32 
func main () {
 n = 3
 fmt.Printf("%T,%v",n,n)
}
>>>
float32,3

    复数类型¶

    package main

import (  
    "fmt"
)

func main() {  
    c1 := complex(5, 7)
    c2 := 8 + 27i
    cadd := c1 + c2
    fmt.Println("sum:", cadd)
    cmul := c1 * c2
    fmt.Println("product:", cmul)
}
